# Infraestrutura de Rede com VirtualBox

Projeto educacional com máquinas virtuais (Debian Firewall, Windows Server, Debian DBA e Windows 10 Cliente) em um ambiente de rede corporativa simulado.

## 📋 Componentes

- **Debian Firewall/Router** - Gerencia internet e protege a rede
- **Windows Server 2012** - DHCP e Active Directory
- **Debian DBA** - Servidor de banco de dados
- **Windows 10 Pro** - Cliente da rede

## 🎯 Objetivo

Demonstrar o funcionamento prático de uma infraestrutura de rede real usando máquinas virtuais.

## 📺 Documentação

- [Diagrama da Topologia](diagramaProjeto.md)

## 🔧 Requisitos

- VirtualBox instalado
- Mínimo 8GB de RAM
- Espaço em disco para 4 VMs
